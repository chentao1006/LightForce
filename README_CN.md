# Light Force (光明原力)

**愿光明原力与你同在，驱散黑暗。**

[English](./README.md) | ***简体中文***

---

Light Force 是一个轻量、简洁的 Chrome 扩展，旨在实现一个高尚的目标：让互联网回归光明。它能够切断网站强制的深色模式 CSS，并强力照明你访问的网站，瞬间将其外观切换回易读、明亮且干净的浅色主题。

## 🌟 为什么选择 Light Force?

*   **对散光用户更友好：** 对于许多患有散光的读者来说，在黑色背景上阅读白色文字会产生“光晕”效应（文字会模糊并渗入背景）。Light Force 通过将对比度瞬间切换回舒适的白底黑字，完美解决这一问题。
*   **夺回视觉主导权：** 不要让网站决定你的视觉体验。如果你的操作系统设置了深色模式，但你希望浏览器标签页保持浅色，网站往往会覆盖你的浏览器设置并强行推送深色页面。Light Force 负责“覆盖这种覆盖”。
*   **日光下浏览的完美拍档：** 深色模式在漆黑的房间里很棒，但当你身处光线充足的办公室、户外工作或坐在窗边时，白色背景能大幅减少屏幕眩光和反射。

## ⚡ 核心功能

*   **原力精准打击：** Light Force 非常智能。它专门针对那些通过 CSS 媒体查询强制呈现深色模式的网站。如果一个网站本身就是浅色主题，Light Force 不会对其进行任何改动。
*   **一键切换：** 通过设计精美的弹出窗口，瞬间开启或关闭扩展。
*   **实时视觉反馈：** 扩展图标带有一个醒目的“ON”标记，当原力激活时会亮起，让你随时掌握当前状态。
*   **轻量且快速：** 采用最新的 Manifest V3 Service Workers 技术构建，运行流畅。尊重用户隐私，处理过程完全在本地完成。

## 🛠️ 安装方法

### Chrome 扩展程序 (开发者模式)

1. 克隆或下载本仓库到本地。
2. 打开 Google Chrome，导航至 `chrome://extensions/`。
3. 开启右上角的 **开发者模式**。
4. 点击 **加载已解压的扩展程序** 按钮，选择 `light-force` 目录。
5. 扩展现已安装！别忘了固定它以便快速访问。

### 用户脚本 (Tampermonkey / Violentmonkey)

1. 确保你已安装脚本管理器，如 [Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/)。
2. 打开本仓库中的 [light-force.user.js](./light-force.user.js) 文件。
3. 脚本管理器应会自动检测并提示安装。
4. 如果没有自动提示，请复制 `light-force.user.js` 的全部内容，在脚本管理器面板中新建脚本并粘贴。
5. 保存脚本，光明原力将伴随你访问所有网站！

## 💻 技术细节

Light Force 利用了最新的 Chrome Extension API (**Manifest V3**)。它通过在 `document_start` 时注入内容脚本，拦截并覆盖 CSS 的 `prefers-color-scheme: dark` 媒体查询，强制浏览器渲染其浅色主题变体。

使用权限：
- `storage`: 用于记住你的开关设置。
- `activeTab`: 允许弹出窗口与当前活动页面交互。
- `<all_urls>` (Content Script): 在所有站点上注入光明原力 CSS 覆盖逻辑。

## 🔒 隐私声明

Light Force 仅在你的浏览器本地修改网站的 CSS 样式以提升易读性。它**不会**收集、跟踪或传输你的任何个人数据、按键记录或浏览历史。
