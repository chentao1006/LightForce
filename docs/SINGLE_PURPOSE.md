### 单一用途 (Single Purpose)

**中文内容 (如果你需要提交中文)：**
本扩展程序的唯一下一用途是：**强制将暗色（Dark Mode）网页转换为亮色（Light Mode）网页**，从而提高特定用户群体（如散光患者或强光环境下的阅读者）的阅读体验，不包含任何与此修改页面主题颜色无关的功能。

**英文内容 (如果你需要提交英文)：**
The single purpose of this extension is to **force dark-themed websites into a light theme (Light Mode)** to improve readability for specific users, such as those with astigmatism or reading in bright environments. It does not contain any supplementary features unrelated to modifying the webpage's color scheme.
